# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built to showcase my skills, projects, internships, certifications, leadership experience, and professional journey as a Full Stack & Java Developer.

## 🚀 Live Demo

Portfolio Website:

```text
https://portfolio-shaik-adil.vercel.app/
```

## ✨ Features

* Modern and responsive UI
* Interactive hero section with video background
* About Me section with profile card
* Technical Skills showcase
* Featured Projects section
* Internship Experience
* Leadership & Extracurricular Activities
* Certifications section
* Contact Information & Social Links
* Mobile-friendly design
* Smooth animations using AOS

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/shaikadil07/portfolio.git
```

Navigate to project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
